My Projects

  Piano Keyboard

I made this piano keyboard in vanilla Javascript based on the dynamically generated synthetic keyboard made by Keith William Horwood. I extended the number of keys available to 4 octaves, set new key bindings, and applied some stylistic changes. I wrote a step-by-step tutorial explaining my code on Free Code Camp. What makes this keyboard interesting is the fact that Javascript dynamically generates the appropriate sound when the user presses a key and the code does not load any external audio files. The keyboard is meant to be played on the computer and therefore it is not responsive.

Technologies used: HTML, CSS, JavaScript
   

  Pomodoro Clock

The Pomodoro Technique is a method in which a timer is used to break down work into intervals, traditionally 25 minutes in length, separated by short breaks. This app allows the user to adjust the work interval via a session length. When the session finishes, the timer will immediately switch to a break length. The timer switches back and forth between session and break until the user hits pause or reset. I originally coded this project using only React and then later refactored the code to manage the application state using Redux (link below).

Technologies used: HTML, CSS (preprocessed with SASS), Bootstrap, React, Redux
   

  Whack-a-Mole Game

I had a lot of fun coding this Whack-a-Mole game in vanilla JavaScript. With the additional use of some CSS animation, I brought to life a classic game in the web browser. Players get 10 seconds to whack as many moles as they can. The order in which moles pop out is of course randomized. Are you ready to test your reflexes?

Technologies used: HTML, CSS, JavaScript
