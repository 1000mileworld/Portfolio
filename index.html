About

Hey, my name's Joe Liang. I have more than a decade of programming experience and transitioned into a freelance front-end web designer/developer. I'm passionate about a variety of web technologies with particular fondness for JavaScript and React. Some of my favorite apps I've made using these technologies include a piano keyboard and pomodoro clock.

In between projects, you can always find me learning and experimenting with new ideas. Currently I'm having fun with some animation libraries such as Animate.css and Anime.js. I also aspire to become a full stack developer, with plans to learn Node.js and SQL in the works. I share my joy of learning web development with others through running a blog on 1000 Mile World and writing beginner-friendly articles on freeCodeCamp.org.

In my spare time, I'm an avid ultimate frisbee player. I enjoy good food, good beer, and good company.

Services

  • Design Website UI: A good user interface is essential for converting visitors into your loyal customer base. I design the website UI to create a beautiful user experience.
  • Code Front-End Site: Front-end development involves building the visual components of a website. Using HTML, CSS, and JavaScript I build from scratch modern, responsive websites that follow current best practices in web development. If you've got a design, I can write the front-end code for it.
  • Set Up WordPress: WordPress is a powerful content management system that allows someone to run a website with no coding experience. However, getting started can still be an arduous task and I can help you set up your website in WordPress, handing over the reins to you when it's ready.
  • Improve Site Performance: As part of a great user experience, having a fast website is also key to converting visitors into potential customers. I use various techniques to improve your site performance so that your pages load fast whether they are viewed on a desktop or mobile device.

These services can be combined or taken as standalones when appropriate.
